## Number

**เลข หรือ Number** เป็นข้อมูลที่เก็บตัวเลข โดยการเก็บเป็นข้อมูลตัวเลขจะสามารถนำมาใช้เครื่องหมายทางคณิตศาสตร์ต่างๆได้ เช่น บวก `+` ลบ `-` คูณ `*` หาร `/` ได้ รวมทั้งเครื่องหมายเปรียบเทียบอย่าง น้อยกว่า `<` มากกว่า `>`

```javascript
let x = 2;
let y = 5;
alert(x + y);
alert(x - y);
alert(x * y);
alert(x / y);
```

## String

**ข้อความ หรือ String** เป็นข้อมูลที่เก็บข้อความ โดยจะสามารถใช้เครื่องหมายเพื่อระบุประเภทข้อมูลเป็น String ได้ดังนี้

- Single-quote `'` เช่น `'Hello world!'`
- Double-quote `"` เช่น `"Hello world!"`
- Backtick <code class="highlighter-rouge">&#96;</code> เช่น <code class="highlighter-rouge">&#96;Hello world!&#96;</code>
  - Backtick เป็นเครื่องหมายที่อยู่ตรงปุ่มเปลี่ยนภาษาที่หลายๆคนใช้กัน

```javascript
let singleQuote = 'Hi!'
  , doubleQuote = "Hello!"
  , backtick = `Hooya!`;
alert(singleQuote);
alert(doubleQuote);
alert(backtick);
```

> **หมายเหตุ:** โดยส่วนมากโปรแกรมเมอร์จะกำหนด String ด้วยเครื่องหมาย Single-quote มากกว่าเครื่องหมายอื่นๆ

## Boolean

**ตรรกะ หรือ Boolean** เป็นข้อมูลที่เก็บเฉพาะ **true (จริง)** และ **false (เท็จ)** เท่านั้น จะถูกนำมาใช้เพื่อตรวจสอบเงื่อนไขต่างๆ

```javascript
let x = 2;
let y = 5;
alert(x < y); // x < y จริง
alert(x > y); // x > y เท็จ
```

## Null, Undefined

ค่าว่างเปล่าจะมี 2 รูปแบบ คือค่าที่ถูกกำหนดให้ว่าง (null) ที่ถูกบังคับให้เป็น null และค่าที่ไม่ได้ถูกกำหนด (undefined) ที่ถูกสร้างชื่อตัวแปรแต่ยังไม่ได้กำหนดค่าใดๆ

```javascript
let x = null;
let y;
alert(x); // แสดงเป็น null เพราะถูกกำหนดเป็น null
alert(y); // แสดงเป็น undefined เพราะไม่ได้ถูกกำหนดค่าใดๆ
```

## การขัดแย้งของข้อมูล

ในภาษาโปรแกรมโดยทั่วไปแล้วจะมีการใช้ประเภทของข้อมูลที่แตกต่างกันออกไป ตามโจทย์ที่ต้องการแก้ เพื่อให้แสดงผลและมีการทำงานอย่างถูกต้องโดยไม่มีการขัดแย้งข้อมูลกันเอง

โดยตัวอย่างการขัดแย้งกันของข้อมูลคณิตศาสตร์ด้วยกันเอง อย่างการนำ `1 / 0` จะไม่สามารถหารได้ ผลลัพธ์จะถูกแสดงเป็น `Infinity` แทน

```javascript
alert(1 / 0); // Infinity
```

และหากนำข้อมูล Number มาหาร `/` กับข้อมูล String จะทำให้ได้ผลลัพธ์เป็น `NaN` หรือก็คือ **Not a Number** เพราะทั้งสองข้อมูลนี้เมื่อใช้เครื่องหมายคณิตศาสตร์กันจะไม่สามารถคำนวณได้

```javascript
alert('Hello world!' / 2); // NaN
```

แต่การนำข้อมูลของ Number บวก `+` กับ String จะทำให้ข้อมูลถูกแปลงเป็น String ทั้งหมด และแสดงผลออกมาเป็นการรวมข้อความเข้าด้วยกัน

```javascript
alert('Hello Test ' + 123); // Hello Test 123
alert(123 + ' Hello Test'); // 123 Hello Test
alert(123 + 123); // 246
alert('123' + 123); // 123123
alert('123' + '123'); // 123123
```

> **เพิ่มเติม:** เพื่อการสังเกตได้ง่ายๆโดยเบื้องต้นสำหรับมือใหม่ คือการที่มีเครื่องหมาย Quote ที่เป็นฟันหนูคู่ (Double-quote) หรือฟันหนูข้างเดียว (Single-quote) และฟันหนูเอียง (Backtick) หากมีเครื่องหมายเหล่านี้คลุมข้อความ นั้นคือ **String** เช่น `'Hello'` หรือ `'123'` ก็คือข้อความเช่นกัน
>
> ขณะที่หากข้อมูลนั้นเป็นข้อความเหมือนกัน แต่ไม่มีการคลุมฟันหนูใดๆเลย นั้นคือ **Variable** เช่น `Hello`
>
> และสุดท้ายหากข้อมูลนั้นเป็นตัวเลข และไม่มีการคลุมฟันหนูใดๆเลย นั้นคือ **Number** เช่น `123`

> **คำเตือน:** ในการใช้ประเภทข้อมูล ควรกำหนดให้ถูกต้องตามที่ต้องการ เพราะภาษา JavaScript เป็นภาษาที่ **Dynamically** ทำให้ประเภทของข้อมูลถูกเปลี่ยนแปลงได้ตลอด ขณะที่โปรแกรมอื่นๆอาจจะไม่มีคุณสมบัตินี้ ทำให้การแปลงข้อมูลค่อนข้างลำบาก ดังนั้นมีจุดประสงค์ใช้เพื่อการคำนวนก็ควรเลือกใช้ **Number** ที่ถูกต้อง หรือใช้ **String** หากต้องการแสดงผล
{:.is-warning}

## Escape

ในกรณีที่ต้องการเครื่องหมายมาใช้ร่วมกับข้อความด้วย เช่น `My name's John` อาจจะทำให้เกิดปัญหาดังตัวอย่างด้านล่าง เพราะมีเครื่องหมายซ้ำกับการเปิดและปิดของ Single-quote

```javascript
alert('My name's John');
```
{:.is-danger}

เราจะเห็นว่าไม่สามารถใช้ได้ เพราะมันมองแค่ `My name` ในขณะนี้ข้อความด้านหลัง `s John` ถูกตัดตอนไปเพราะเครื่องหมาย Single-quote ซ้ำกัน ดังนั้นการใช้ Double-quote หรือ Backtick จึงสามารถช่วยด้านนี้ได้ หรือสามารถใช้ `\` นำหน้าเครื่องหมาย เพื่อทำการ **Escape** ข้อความ ทำให้ไม่ชนกันของเครื่องหมาย

```javascript
alert("My name's John 1"); // ใช้ Double-quote
alert(`My name's John 2`); // ใช้ Backtick
alert('My name\'s John 3'); // ใช้ Escape ด้วย \'
```

และนอกจาก `\'` ที่ทำหน้าที่แสดงผลของ Single-quote แล้ว ยังมี Escape อีกหลายๆตัว เช่น `\n` คือการใช้เพื่อเว้นบรรทัด ดังตัวอย่าง

```javascript
alert('Oh, Hello!\nWelcome to our project');
alert('Create new five lines.\n\n\n\n\nHi, Five!');
```

> **ยกตัวอย่าง:** อธิบายให้เข้าใจง่ายๆคือ **Escape** แปลได้ว่า **หนี** โดยมันเป็น String พยายาม **หนีจากหน้าที่** จากเดิม เช่น `\n` แทนที่จะแสดงเป็นข้อความธรรม กลับหนีหน้าที่กลายเป็นการเว้นวรรคแทน เป็นต้น

## การจัดรูปแบบข้อความ

ในการจัดรูปแบบข้อความมีหลายวิธี ยกตัวอย่างจากโจทย์ที่แล้วที่เราได้เมื่อบท Variables โดยเราจะทำการจัดรูปแบบของตัวแปรให้แสดงเป็นข้อความมากขึ้น

```js
let firstname = 'John'
  , lastname = 'Doe'
  , age = 18;
```

เราจะนำตัวแปรเหล่านี้มาจัดรูปแบบ เพื่อการแสดงผลข้อความโดยเบื้องต้น โดยเราสามารถใช้ `+` ในการรวมข้อความได้

```javascript
let firstname = 'John'
  , lastname = 'Doe'
  , age = 18;
alert('My name is ' + firstname + ' ' + lastname + ', I\'m ' + age);
// Output: My name is John Doe, I'm 18
```

แต่เราอาจจะเห็นว่าการจัดข้อความด้วยการ `+` String นั้นค่อนข้างยุ่งยาก เพราะต้องจัดตำแหน่ง `'` ให้ถูกต้อง ไม่เช่นนั้นจะทำให้โปรแกรมแบ่งข้อความไม่ถูกต้องได้ ดังนั้นจะมีการใช้ <code class="highlighter-rouge">&#96;</code> ที่เป็นเครื่องหมาย Backtick จะมีคุณสมบัติเป็น **Template** ที่สามารถจัดตำแหน่งของตัวแปรโดยไม่ต้องใช้ `+` ได้โดยการเรียกตัวแปร `${variable}` ภายในเครื่องหมาย แถมเครื่องหมาย ฺBacktick เองก็ไม่ซ้ำกับ Quote อื่นๆที่อาจจะเรียกใช้ประจำอีกด้วย

```javascript
let firstname = 'John'
  , lastname = 'Doe'
  , age = 18;
alert(`My name is ${firstname} ${lastname}, I'm ${age}`);
// Output: My name is John Doe, I'm 18
```

นอกจากนี้การใช้ Backtick จะช่วยให้การจัดข้อความสามารถเว้นบรรทัดได้โดยไม่จำเป็นต้องใช้ `\` เช่น

```javascript
alert('Hello, \
Work!');
```

```javascript
alert('Hello,
Error!');
```
{:.is-danger}

เพื่อป้องกันการ Error จากการเว้นบรรทัดของภายใน Quote จะต้องใช้ Escape ลงท้ายเพื่อให้ String ยังคงต่อไปยังอีกบรรทัด โดยส่วนมากใช้ทริคนี้เพื่อสิ่งนี้

```javascript
alert('Hello!\
\nWork with new line!\
\nAnd another line!');
```

แต่การใช้ Backtick จะทำให้สามารถเว้นบรรทัดได้โดยไม่เกิด Error แถมยังไม่จำเป็นต้อง Escape `\n` เพื่อทำการเว้นบรรทัดอีกด้วย

```javascript
alert(`Hello
Work with new line!
And another line!`);
```

> **หมายเหตุ:** เนื่องจาก Backtick อยู่บนตำแหน่ง **ปุ่มเปลี่ยนภาษา (Grave Accent)** ดังนั้นวิธีการพิมพ์เครื่องหมาย <code class="highlighter-rouge">&#96;</code> อาจจะเป็นเรื่องยากสักหน่อยเพราะเมื่อกดแล้วจะไม่มีอะไรเกิดขึ้น ก็สามารถทำได้โดยกดปุ่ม `Alt + 96` กล่าวคือกด `Alt` ค้างไว้ จากนั้นกด `9` แล้วปล่อย และกด `6` จากนั้นปล่อย และปล่อย `Alt` เป็นขั้นตอนสุดท้าย จะเห็นว่าเราเพิ่งได้พิมพ์เครื่องหมาย Backtick ออกมา
>
> ซึ่งแน่นอนว่าค่อนข้างเป็นเรื่องยุ่งยากหากเริ่มใช้ Template บ่อยครั้ง ดังนั้นตั้งปุ่มเปลี่ยนภาษาเป็น `Alt + Shift` หรือ `Win + Spacebar` จะช่วยการกดปุ่ม Backtick เป็นไปได้ง่ายขึ้น

## typeof()

`typeof()` เป็นคำสั่งสำหรับการแสดงให้ทราบว่าเป็นข้อมูลประเภทใด

```javascript
let x = 20; // กำหนดค่า x = 20
let showType = typeof(x); // กำหนดค่า showType = typeof(x)
alert(showType); // showType = 'number'
```

โดยในขั้นตอนนี้เราอาจจะยุบรวมเขียนให้สั้นลง โดยไม่จำเป็นต้องประกาศตัวแปรหลายๆตัว และผสมกับการใช้ Template เพื่อแสดงตัวอย่างให้ชัดเจนมากขึ้น ดังตัวอย่างด้านล่าง

```javascript
alert(`20 is ${typeof(20)}`);
alert(`1 / 0 is ${typeof(1 / 0)}`); // ตามปกติจะได้เป็น Infinity แต่นั้นก็เป็นประเภทข้อมูลแบบ number
alert(`20 / 'string' is ${typeof(20 / 'string')}`); // ตามปกติจะได้เป็น NaN แต่นั้นก็เป็นประเภทข้อมูลแบบ number
alert(`undefined is ${typeof(undefined)}`);
alert(`true is ${typeof(true)}`);
alert(`'foobar' is ${typeof('foobar')}`);
```

> **หมายเหตุ:** `typeof()` สามารถใช้ได้สองแบบ คือทั้งแบบมี และไม่มีวงเล็บ `()` แต่จะต้องเว้นวรรค ดังตัวอย่าง โดยเราจะเลือกใช้ `typeof()` เพื่อกันความสับสน
>
> ```javascript
> alert(typeof(20));
> alert(typeof 20);
> ```

## บทสรุป

- การจำแนกชนิดของข้อมูล โดยเป็นชนิดพื้นฐาน แต่จะยังมีชนิดอื่นๆเพิ่มเติมด้วย
  - Number
  - String
  - Boolean
  - Null
  - Undefined
- เข้าใจวิธีการขัดแย้งของข้อมูลต่างชนิด ทำให้เกิดการแปลงชนิดข้อมูล หรือมีความผิดพลาดได้
- การใช้ Escape เพื่อการใช้คำสั่งพิเศษบางอย่างผ่าน String เช่น เว้นวรรค `\n` หรือเพื่อแสดงผล Single-quote ไม่ให้ชนกับตัวแบ่งข้อความ `\'`
- การจัดรูปแบบข้อความด้วยเครื่องหมาย Backtick <code class="highlighter-rouge">&#96;</code>
- การใช้ `typeof()` ในการบอกชนิดของข้อมูล
